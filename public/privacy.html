2. Keine Datenerhebung

VBA BEAST ist eine 100% lokale Offline-App. Alle Daten (Excel-Dateien, Word-Dokumente, Einstellungen, Makros, Pipelines) werden ausschließlich auf Ihrem Gerät gespeichert — in IndexedDB und localStorage Ihres Browsers.

Es werden keine personenbezogenen Daten an einen Server übertragen.

3. Lokale Speicherung

  • IndexedDB: Excel-Daten, Workspaces, Makros, Pipelines, Dokumente, Templates
  • localStorage: Theme-Einstellung, Seitennavigation, Entitlements (PRO-Status)

Sie können alle gespeicherten Daten jederzeit über die Browser-Einstellungen oder die App-Funktion "Daten löschen" entfernen.

4. Optionale AI-Features

Falls Sie die optionalen AI-Features nutzen (Workflow Builder, Daten-Profiler, Formel-Assistent), werden Datenausschnitte an die Claude API (über unseren Proxy-Server) gesendet. Dabei gilt:

  • Nur die für die AI-Anfrage notwendigen Spalten-Samples werden gesendet
  • Keine vollständigen Datensätze oder Dateien
  • Keine Speicherung auf dem Server (Proxy leitet weiter und verwirft)
  • BYOK (Bring Your Own Key): Ihr API-Key wird nur lokal gespeichert

5. In-App-Käufe

Zahlungen werden ausschließlich über Apple App Store, Google Play Store oder Stripe abgewickelt. Wir haben keinen Zugriff auf Ihre Zahlungsdaten.

6. Keine Cookies, kein Tracking

VBA BEAST verwendet keine Cookies, kein Analytics, kein Tracking. Es werden keine Daten an Dritte weitergegeben.

7. Ihre Rechte (DSGVO Art. 15-21)

Da keine personenbezogenen Daten erhoben werden, fallen die Rechte auf Auskunft, Löschung und Berichtigung praktisch nicht an. Sollten Sie dennoch Fragen haben, kontaktieren Sie uns.

8. Änderungen

Diese Datenschutzerklärung kann bei App-Updates angepasst werden. Die aktuelle Version ist immer in der App und auf der Website einsehbar.
